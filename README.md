Projeto Agrinho 2026 – Terra Indígena Queimadas
Sobre o projeto

Este site foi desenvolvido como trabalho escolar para o Projeto Agrinho 2026. A proposta da página é apresentar informações sobre a comunidade onde vivem os estudantes: a Terra Indígena Queimadas, localizada no município de Ortigueira, no estado do Paraná.

O projeto busca valorizar a comunidade, a escola, o território, a cultura, a natureza e os modos de vida locais. Além disso, o trabalho também tem como objetivo desenvolver habilidades relacionadas ao uso da tecnologia, à criação de páginas web e à publicação de projetos na internet por meio do GitHub.

A página foi pensada para ser simples, organizada e acessível, permitindo que os alunos apresentem sua realidade de forma respeitosa, educativa e visualmente agradável.

Objetivo do site

O principal objetivo do site é mostrar a importância da Terra Indígena Queimadas, destacando a comunidade, a escola, a relação com a natureza, a cultura e a valorização do território.

O site também tem como finalidade incentivar os estudantes a aprenderem conceitos básicos de desenvolvimento web, utilizando HTML, CSS e JavaScript na construção de uma página publicada no GitHub Pages.

Tema escolhido

O tema escolhido foi a comunidade Terra Indígena Queimadas, pois ela faz parte da realidade dos estudantes e representa o lugar onde vivem, estudam, convivem e constroem suas histórias.

A escolha desse tema permite que os alunos relacionem o aprendizado de tecnologia com a valorização da própria comunidade, fortalecendo a identidade local e o respeito ao território.

Tecnologias utilizadas

Para desenvolver o site, foram utilizadas as seguintes tecnologias:

HTML5: utilizado para criar a estrutura da página.
CSS3: utilizado para organizar o visual, as cores, os tamanhos, os espaçamentos e a responsividade do site.
JavaScript: utilizado para criar interações, como o menu responsivo e a ampliação das imagens ao clicar.
GitHub: utilizado para armazenar os arquivos do projeto.
GitHub Pages: utilizado para publicar o site na internet.

Arquivos do projeto

O projeto é formado pelos seguintes arquivos e pastas:

index.html: arquivo principal do site, onde está a estrutura da página.
style.css: arquivo responsável pela aparência do site, como cores, fontes, imagens, tamanhos e organização das seções.
script.js: arquivo responsável pelas interações do site.
README.md: arquivo com as informações sobre o projeto.
img: pasta onde ficam armazenadas as imagens utilizadas no site.

Estrutura do site

O site foi organizado em diferentes seções para facilitar a navegação e a leitura. As principais partes da página são:

Início: apresentação principal do site.
Nossa comunidade: informações sobre a Terra Indígena Queimadas.
Cultura e identidade: valorização dos saberes, costumes e formas de vida da comunidade.
Natureza e sustentabilidade: reflexão sobre o cuidado com a natureza e o território.
Nossa escola: apresentação da escola da comunidade, com nome e imagem.
Galeria da comunidade: espaço para imagens autorizadas da comunidade, da escola, da natureza e do território.
Projeto Agrinho 2026: explicação sobre a relação do site com o projeto escolar.
Passos utilizados para concluir a página

Para concluir a criação do site, foram seguidos alguns passos principais.

Primeiramente, foi definido o tema do projeto. A turma escolheu trabalhar com a comunidade Terra Indígena Queimadas, por ser o local onde os estudantes vivem e por representar um espaço importante de identidade, cultura, aprendizagem e convivência.

Depois, foi planejada a estrutura da página. Foram escolhidas as seções que fariam parte do site, como comunidade, cultura, natureza, escola, galeria de fotos e explicação sobre o Projeto Agrinho 2026.

Em seguida, foi criado o arquivo index.html, responsável por organizar todo o conteúdo da página. Nesse arquivo foram inseridos os títulos, textos, menus, imagens e divisões do site.

Após a criação da estrutura, foi desenvolvido o arquivo style.css, utilizado para melhorar a aparência da página. Nele foram definidas as cores, os espaçamentos, o tamanho das imagens, o formato dos cards, a organização da galeria e a adaptação do site para celulares.

Também foi criado o arquivo script.js, responsável por adicionar interatividade à página. Com JavaScript, foi possível criar o menu para telas menores e permitir que as imagens fossem ampliadas quando clicadas.

Depois, foi criada a pasta img, onde foram colocadas as imagens utilizadas no site, como a foto da escola e as imagens da galeria. Para que as imagens aparecessem corretamente, os caminhos foram ajustados no HTML, utilizando a forma img/nome-da-imagem.jpg.

Por fim, os arquivos foram organizados para publicação no GitHub. O projeto foi preparado para ser enviado a um repositório e publicado por meio do GitHub Pages, permitindo que o site pudesse ser acessado pela internet.

Como visualizar o site

Para visualizar o site no computador, basta abrir o arquivo index.html em um navegador de internet.

Para publicar o site no GitHub Pages, siga os passos:

Criar uma conta ou acessar uma conta no GitHub.
Criar um novo repositório.
Enviar os arquivos do projeto para o repositório.
Verificar se os arquivos index.html, style.css, script.js, README.md e a pasta img foram enviados corretamente.
Acessar a opção Settings do repositório.
Entrar em Pages.
Selecionar a branch principal, geralmente chamada main.
Salvar a configuração.
Aguardar o GitHub gerar o link do site.
Cuidados com as imagens e informações

As imagens utilizadas no site devem ter autorização para publicação. É importante evitar a exposição de pessoas, crianças, espaços internos, práticas culturais, rituais, histórias, cantos, conhecimentos tradicionais ou informações da comunidade sem autorização dos responsáveis, professores, famílias e lideranças.

O site deve valorizar a comunidade sem divulgar informações sensíveis ou conteúdos que não devem ser compartilhados publicamente.

Conclusão

A criação deste site possibilitou a união entre tecnologia, educação e valorização da comunidade. Durante o desenvolvimento do projeto, os estudantes aprenderam a organizar informações, estruturar uma página web, trabalhar com imagens, aplicar estilos visuais e utilizar recursos de interatividade.

Além de desenvolver conhecimentos sobre HTML, CSS, JavaScript e GitHub, o projeto também contribuiu para fortalecer o olhar dos estudantes sobre a importância da Terra Indígena Queimadas, da escola, da cultura, da natureza e do território onde vivem.
